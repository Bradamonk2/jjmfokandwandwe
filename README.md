JJ Mfoka Ndwandwe — Official Media Profile
A professional, single-file HTML media profile for JJ Mfoka Ndwandwe, rising Maskandi artist from Nongoma, KwaZulu-Natal. Built for sharing with media houses, journalists, and booking agents — and deployable to the web in minutes.

📁 File
FileDescriptionindex.htmlThe complete media profile — all styles, images, and logic in one self-contained file

Note: The file is named index.html for web deployment. Vercel and all static hosting platforms serve index.html as the homepage automatically.


🌐 Deploying to Vercel (Free)
Option A — Drag & Drop (Quickest)

Go to vercel.com and sign up free (GitHub login recommended)
From your dashboard click "Add New Project"
Choose "Deploy from your computer" and drag index.html into the upload area
Click Deploy
Your live URL will be ready in ~30 seconds — e.g. jj-mfoka.vercel.app

Option B — Via GitHub (Recommended for updates)

Create a free GitHub account
Create a new repository (e.g. jj-mfoka-profile)
Upload index.html to the repository
Go to vercel.com → "Import Git Repository" → select your repo
Click Deploy


Every time you update index.html on GitHub, Vercel will automatically redeploy — no manual steps needed.

Custom Domain (Optional)
Once deployed on Vercel, go to your project's Settings → Domains to connect a custom domain like jjmfokandwandwe.com. Vercel provides free SSL automatically.
